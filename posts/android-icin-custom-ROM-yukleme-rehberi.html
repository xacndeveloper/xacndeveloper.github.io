🛠️ Hazırlık ve Önlemler

Gereken Yazılımlar ve Kurulum (özet)

Windows (yönerge — komut değil):

  1. Platform-Tools paketini indirin: Platform-Tools — Android Developers
  2. ZIP'i açın ve platform-tools klasörünü C:\ sürücüsüne taşıyın (örnek: C:\platform-tools).
  3. Sistem ortam değişkenlerine C:\platform-tools yolunu ekleyin (Denetim Masası → Sistem → Gelişmiş sistem ayarları → Ortam Değişkenleri).
  4. Komut İstemi/PowerShell'i kapatıp tekrar açın; adb version ve fastboot --version ile doğrulayın.

Linux (terminal komutu):

sudo apt update && sudo apt install android-tools-adb android-tools-fastboot

macOS (Homebrew ile):

brew install android-platform-tools

Not: Windows için ADB/Fastboot paketini PowerShell veya CMD üzerinden doğrudan paket yöneticisi ile yükleme adımı yoktur; yukarıdaki yönergeyi manuel uygulayın.

Ön Hazırlık Kontrol Listesi

  • Cihaz şarjı en az %70.
  • Tüm kişisel veriler yedeklendi (fotoğraflar, dosyalar, kişiler).
  • Uygulama yedekleri alındı (Titanium Backup, Swift Backup).
  • Orijinal firmware indirildi (örn. SamMobile, Xiaomi Firmware Updater).
  • USB sürücüleri yüklendi (üretici sitesinden).
  • Model kodu doğrulandı (Ayarlar → Telefon hakkında → Model numarası).
  • USB hata ayıklama etkinleştirildi.
  • OEM kilidi (Geliştirici seçeneklerinde) etkinleştirildi (gerekliyse).

Geliştirici Seçeneklerini Etkinleştirme

  1. Ayarlar → Telefon hakkında → Yapım numarası'na 7 kez dokunun.
  2. Geri dönün → Sistem → Geliştirici seçenekleri.
  3. USB hata ayıklamayı etkinleştirin.
  4. OEM kilidini etkinleştirin (varsa).


🔓 Bootloader Kilidi Açma

Genel Adımlar

  1. Cihazı USB ile bilgisayara bağlayın.
  2. Komutları çalıştırın:
adb devices        # cihazın bağlandığını doğrulayın
adb reboot bootloader
  1. Bootloader modunda:
fastboot devices   # fastboot ile göründüğünü doğrulayın
fastboot flashing unlock
# veya bazı cihazlarda:
fastboot oem unlock
  1. Cihaz fabrika ayarlarına dönebilir; işlemi onaylayın ve cihazı yeniden başlatın.

Markaya Özel Notlar

  • Samsung: Download Mode (+Odin) kullanılır. Knox tetiklenirse bazı güvenlik fonksiyonları kalıcı olarak devre dışı kalır.
  • Xiaomi: Resmi Mi Unlock Tool gerekir; hesap ve bekleme süresi (7–15 gün) istenebilir.
  • Google Pixel: fastboot flashing unlock genelde doğrudan çalışır.
  • OnePlus: fastboot oem unlock veya fastboot flashing unlock.
  • Huawei (eski modeller): Resmi unlock kodları/servisleri zamanla değişti — güncel bilgi kontrol edin.


📲 Custom Recovery Kurulumu

Popüler Recovery'ler

  • TWRP (Team Win Recovery Project) — en yaygın.
  • OrangeFox — modern arayüz, ekstra özellikler.
  • PitchBlack — tema desteği.

Nereden Bulunur

  • Resmi TWRP: TWRP — Team Win
  • XDA Developers: XDA Developers Forum
  • Topluluk kanalları (Telegram/Reddit) — doğruluğunu kontrol edin.

Kurulum Adımları

  1. Cihaza uygun twrp.img dosyasını indirin.
  2. Fastboot'a geçin:
adb reboot bootloader
  1. Recovery yazdırma veya geçici başlatma:
fastboot flash recovery twrp.img
# Geçici çalıştırmak için:
fastboot boot twrp.img
  1. İlk açılışta TWRP'de "Allow System Modification" veya benzeri izinleri verin.


💾 Yedekleme İşlemleri

TWRP ile Tam Yedek (Nandroid)

  1. TWRP → Backup.
  2. Yedeklenecek bölümler: Boot, System, Data, EFS (ÇOK ÖNEMLİ — IMEI), gerekirse Persist/Modem.
  3. Swipe to Backup ile işlemi başlatın.
  4. Yedekleri harici SD karta veya bilgisayara kopyalayın.

EFS / IMEI Yedekleme (örnek)

adb shell
su
dd if=/dev/block/bootdevice/by-name/efs of=/sdcard/efs-backup.img
exit
adb pull /sdcard/efs-backup.img
Not: EFS yolu cihaza göre değişebilir. Cihazınıza özel yolu araştırın; yanlış dd kullanımı tehlikelidir.

Kişisel Veri Yedekleme

  • SMS/Çağrı: SMS Backup & Restore.
  • Kişiler: Google hesabına senkronize edin.
  • Fotoğraflar/dosyalar: Google Drive veya bilgisayara kopyalama.
  • Uygulama verileri: Titanium Backup (root gerektirir) veya Swift Backup.


🔄 Custom ROM Kurulumu

Popüler ROM Kaynakları

  • LineageOS — LineageOS Downloads
  • Pixel Experience — Pixel Experience Downloads
  • ArrowOS — ArrowOS
  • crDroid — crDroid
  • XDA — XDA Developers Forum (cihaz tabanlı, resmi olmayan derlemeler)

GApps (Google Uygulamaları)

  • OpenGApps: OpenGApps (platform, Android sürümü ve variant seçimi önemli).
  • Alternatifler: FlameGApps, NikGApps.
  • Öneri: Başlangıç için nano veya pico varyantı.

Clean Flash (Temiz Kurulum) Adımları

  1. ROM, GApps ve (isteğe bağlı) Magisk zip'lerini cihaza veya SD karta kopyalayın.
  2. TWRP'yi başlatın.
  3. Wipe → Advanced Wipe: Dalvik/ART Cache, System, Data, Cache seçin.
  • Not: Eğer Data şifreliyse (encrypted), şifreyi kaldırmanız veya uygun adımları uygulamanız gerekir.
  1. Install → ROM zip → Swipe to confirm flash.
  2. Hemen ardından GApps (varsa) flashlayın.
  3. Magisk (root isteniyorsa) son olarak flashlanabilir.
  4. Wipe cache/dalvik yapın.
  5. Reboot System.

Dirty Flash (Veri Silmeden Güncelleme)

  • Küçük güncellemelerde işe yarar; major sürümler için clean flash önerilir.

İlk Açılış

  • İlk açılış 5–15 dakika alabilir. Sabırlı olun. Bootloop olursa recovery'den cache temizliği veya ROM yeniden flash gerekebilir.


🛡️ Root ve Modüller

Magisk ile Root

  • Magisk: Magisk GitHub Releases

TWRP ile doğrudan zip flashlama veya boot image patch yöntemi kullanılabilir.

Boot image patch yöntemi (örnek)

# Örnek genel adımlar (cihaz özelinde değişir):
adb pull /dev/block/bootdevice/by-name/boot boot.img
# Telefon üzerinde Magisk uygulaması ile boot.img'i patch'leyin
# Ardından patched image'i fastboot ile flashlayın:
fastboot flash boot magisk_patched.img
Not: Gerçek blok yolları cihaz bazında farklılık gösterir. Yanlış blok seçimi cihazı bozabilir.

Önemli Modüller ve Ayarlar

  • Magisk Hide / DenyList: Root tespitini engellemeye yardımcı (bankacılık uygulamaları için).
  • Modüller: BusyBox, Viper4Android (ses iyileştirme), SafetyNet-fix modülleri.
  • SafetyNet: Banka ve DRM uygulamaları için test edin; her ROM/Magisk sürümünde farklı sonuçlar olabilir.


✔️ Artılar ve ❌ Eksiler

✅ Artıları

  • Yeni Android sürümleri üretici desteği bitmiş cihazlarda kullanılabilir.
  • Performans ve pil ömrü genelde iyileşebilir.
  • Gelişmiş özelleştirme imkânı.
  • Gereksiz OEM uygulamalarından arınma.

❌ Eksileri

  • Garanti kaybı; bazı durumlarda (Knox) geri dönüş yok.
  • Resmi olmayan yazılımlar güvenlik riski taşıyabilir.
  • Donanım uyumluluğu sorunları (kamera, parmak izi vb.).
  • Banka uygulamaları ve DRM sorunları.
  • Sürekli manuel bakım gerektirebilir.


🆘 Sorun Giderme

Bootloop (sürekli yeniden başlatma)

  • Recovery → Wipe cache/dalvik.
  • ROM'u tekrar flashlayın.
  • Önceki Nandroid yedeğini geri yükleyin.

EFS / IMEI kaybı

  • Önlem: mutlaka EFS yedeği alın.
  • Çözüm: EFS yedeğini geri yükleyin veya orijinal firmware ile tam flash atın.
  • Bazı durumlarda servis gerekebilir.

Donanım sorunları (kamera, sensör)

  • Google Camera (GCam) portları deneyin.
  • Farklı ROM veya kamera HAL'ı deneyin.

Banka uygulamaları çalışmıyor

  • Magisk DenyList / Hide seçeneklerini kullanın.
  • SafetyNet-fix modüllerini deneyin.
  • Gerekirse uygulamayı farklı bir profil/sandbox'ta çalıştırma gibi yöntemler araştırın (güvenlik riski olabilir).

Wi-Fi / Bluetooth sorunları

  • Doğru modem/firmware dosyalarını kullanarak yeniden flashlayın.
  • persist ve modem yedeklerini geri yükleyin.


📝 Markaya Özel Rehberler

Samsung

  • Knox tripping: tetiklenirse bazı hizmetler kalıcı kaybolabilir (Secure Folder, Samsung Pay).
  • Odin: Windows için resmi flash aracı (tar/md5 dosyaları).
  • Download Mode kombinasyonları cihaz modeline göre değişir.

Xiaomi

  • Mi Unlock Tool genelde zorunlu; hesap/haftalar arası bekleme olabilir.
  • Anti-Rollback: Eski sürümlere dönme kısıtları olabilir — dikkatli olun.

Google Pixel

  • Bootloader açmak genelde kolaydır (fastboot flashing unlock).
  • Pixel Experience gibi ROM'lar Pixel cihazlara iyi uyum sağlar.

OnePlus

  • Topluluk desteği güçlü; bootloader açma genelde kolaydır.

Eski Huawei

  • Yeni modellerde bootloader açma kısıtlı; eski modellerde resmi kod bulunabiliyordu.
  • Bazı servisler/araçlar ücretli olabilir (örn. DC-Unlocker).


🔒 Güvenlik ve Güncelleme

Güvenlik Önlemleri

  • Sadece güvenilir kaynaklardan ROM/GApps indirin (Lineage, resmi XDA derlemeleri).
  • İndirilen dosyaların MD5/SHA256 hash'lerini kontrol edin.
  • Cihaz şifrelemesini etkinleştirin (ROM destekliyorsa tekrar şifreleyin).
  • Düzenli yedek alın.

Güncelleme Yönetimi

  • Bazı ROM'lar OTA desteği sunar (LineageOS, Pixel Experience).
  • Manual güncelleme: Dirty flash küçük güncellemeler için, major güncellemeler için clean flash önerilir.
  • GApps uyumluluğunu ROM güncellemesi sonrası kontrol edin.


📞 Acil Durum Prosedürleri

Hard Brick (cihaz açılmıyor)

  • Samsung: Download Mode + Odin ile kurtarma (duruma göre).
  • Qualcomm cihazlar: EDL Mode (Emergency Download Mode) ile kurtarma (riskli ve yetki gerekebilir).
  • Kurtarılamazsa profesyonel servis gerekebilir.

Soft Brick (bootloop)

  • Recovery → Nandroid yedeğini geri yükleyin.
  • Fastboot ile factory image flashlamayı deneyin.
  • Cache/Dalvik temizliği yapın.

IMEI kaybı

  • EFS yedeğini geri yükleyin.
  • Yedeğiniz yoksa servis merkezi ile iletişime geçin (yasal düzenlemelere dikkat).
